## WoongDoList -투두리스트-
[WoongDoList](https://woongdolist.netlify.app/)

- 시작

npm start

- 사용된 기술

  - React.js
  - ES6
  - HTML/CSS
  - tailwind
 
- 프로젝트 개요

리액트와 tailwind 환경 학습을 위한 간단한 todo list 사이트입니다!

- 기능
  - 리스트 작성
  - 리스트 삭제
  - 리스트 위치 변경
